== Description ==

With this plugin you'll be able to protect your WordPress login page against theft of passwords.

Colobe Security plugin allows you to use [Colobe Protection Service](https://colobe.net/ "Colobe") in your Wordpress login page, 
in order to prevent brute force attacks.

Colobe Protection Service protects your site against brute-force attacks. 
It has a dynamic list of malicious users, which is real time updated to guarantee the highest level of protection for your site.

Rather than only protecting from brute force attacks, Colobe prevents them. 
It uses its dynamic list to identify maliciuos users during the login process to block the login attempt.

To use the plugin, you need a [Colobe.net](https://colobe.net/ "Colobe") account.

For more information, check out [Colobe.net](https://colobe.net/ "Colobe").

== Installation ==

1. Install Colobe Security by uploading the files to your server
1. Go to colobe.net and create a new free account, then add your site to your account
1. Download the Colobe Library file from your account and upload it with the control panel of this plugin
1. Finish :)
